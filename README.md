## 🎨 Design Principles

### Color Coding System
- **Video modality:** Red/coral tones (#FF6B6B)
- **Audio modality:** Teal/cyan tones (#4ECDC4)
- **Text/PHQ-9:** Yellow/gold tones (#FFE66D)
- **SCID-5-PD:** Mint/green tones (#95E1D3)
- **Crisis indicators:** Red (#E74C3C)
- **Normal/safe:** Green (#2ECC71)

### Typography
- **Headers:** Bold, 11-14pt
- **Labels:** Bold, 9-10pt
- **Data values:** Bold where emphasis needed
- **Annotations:** 8-9pt, italic for secondary info

### Layout Standards
- Consistent spacing and alignment
- Black edges for all shapes (clarity)
- White or colored backgrounds (readability)
- Adequate white space (not cluttered)
